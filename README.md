## 🎮 프로젝트 경험

### 🎧 게임 기반 음악 추천 웹사이트 (Game-based Music Recommendation Web App)
> **React & JavaScript 기반 개인 프로젝트**

- **기간:** 2025.06 ~ 2025.11  
- **소개:** 사용자의 게임 플레이 취향을 분석하여 음악을 추천하는 웹 애플리케이션  
- **주요 기능:**  
  - 게임 장르 선택 → 관련 음악 리스트 추천  
  - Spotify API를 활용한 실시간 음악 데이터 연동  
  - React Router를 통한 SPA 구조 구성  
  - 반응형 UI 구현 (Desktop / Mobile 대응)  
- **역할:** 프론트엔드 전반 개발, UI 설계, API 연동  
- **성과:**  
  - 사용자 경험(UX)을 고려한 인터페이스 개선  
  - GitHub Pages를 통한 배포 및 프로젝트 관리  
- **GitHub Repo:** https://github.com/rainbreeze/capstone-server, https://github.com/rainbreeze/capstone-client
- ** URL 주소:** beatopia.site
